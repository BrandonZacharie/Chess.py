Chess.py
===
A simple Chess engine.

## Develop

### Setup
Create virtual environment.
```bash
pip install virtualenv
virtualenv .venv
```

Install packages.
```bash
make install
```

### Build
Run the game.
```bash
make run
```

### Test
Run all tests.
```bash
make test
```

Generate coverage reports.
```bash
make report
```

### Commit
Format code.
```bash
make format
```
